## Setup
- Add `wpapi-js` dependency using yarn or npm to your project
- Add `wpapi-js` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    'wpapi-js',

    // With options
    ['wpapi-js', { /* module options */ }],
 ]
}
```
